# Emmy-love-api

A script to generate [LÖVE](https://love2d.org/) API autocomplete files for [EmmyLua](https://github.com/EmmyLua/IntelliJ-EmmyLua), based on [Emmy-love-api](https://github.com/EmmyLua/Emmy-love-api), that will generate more complete API than the original one.

## How to use it

1. Clone this repository, and init the submodules
2. Create a directory named `api` in the directory, do not put any files in there or run anything in it.
3. Run `lua genEmmyAPI.lua` in the directory

Once you start or refresh your IDE (might be automatic) you should have autocomplete and quick documentation for LÖVE!
